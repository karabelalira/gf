1. открывать в консоли файл для редактирования нужно через vi <имя файла>
vi .gitignore

2. вписываем туда файлы и папки, которые нужно игнорировать

# Compiled gf files #
###################
*.gfo

# Resource folders #
###################
Food/ 


3. закрываем файл и сохраняем изменения :wq
 Exit Vi:                       :q
 Exit Vi (ignore changes):      :q!
 Save:                          :w
 Save and Exit:                 :wq
